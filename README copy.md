# Treesyt

## Run Locally

Clone the project

```bash
  git clone git@github.com:treesyt/treesyt_web_frontend.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
